# Shirt Shop

A modern e-commerce website for selling shirts and apparel, built with Next.js.

## Prerequisites

You'll need [Node.js](https://nodejs.org/) (LTS version recommended) installed on your system. This includes npm, which you'll use to manage dependencies.

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see your site.

## Available Commands

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm start` — Run production server
- `npm run lint` — Run ESLint

## File Structure

- `app/` — Pages and layouts (Next.js App Router)
- `app/globals.css` — Global styles with Tailwind
- `package.json` — Dependencies and scripts
